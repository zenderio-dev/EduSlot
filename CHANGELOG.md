Initial EduSlot release.

- Domain models for workload, preferences, schedule items and conflicts.
- Teacher preference parser.

- Alternative schedule variants.

- Public Python API.
- CLI commands.
- Demo command.
- Streamlit demo interface.
- Sphinx Markdown documentation with Mermaid diagrams.
- Package metadata for PyPI publication.
